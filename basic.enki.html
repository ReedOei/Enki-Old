 86 -- We may define data types (and optionally specify types).
 87 a Point has: (X : Int, and Y : Int).
 88 
 89 -- We may define functions over our custom types
 90 the distance from the origin to Point is:
 91     the square root of ((X of Point)^2 + (Y of Point)^2).
 92 
 93 -- We may similarly define sum types:
 94 a Tree (containing A) may be:
 95     Leaf has: V : A.
 96     Tree has: (V : A, and LeftChild : Tree (containing A), and RightChild : Tree (containing A)).
 97 
 98 -- Then we may define rules for our types:
 99 the height of Tree is:
100     when Tree = Leaf has: V then: 0.
101     when Tree = Tree has: (V, LeftChild, RightChild) then:
102         the height of Tree = 1 + the greater of (the height of LeftChild) (the height of RightChild).
103 
104 the sum of the nodes in Tree is:
105     when Tree = Leaf has: V then: V.
106     when Tree = Tree has: (V, LeftChild, RightChild) then:
107         V + the sum of the nodes in LeftChild + the sum of the nodes in RightChild.
108 
109 -- We may create type classes (interfaces) and add our types to them:
110 A is a type of: Monoid if it has:
111     an identity : -> A.
112     A + B : A -> A -> A.
113 
114 a Point is a type of: Monoid because it has:
115     an identity is: Point has: (X = 0, and Y = 0).
116 
117     A + B is:
118         when (A = Point has: (X1, Y1), and
119               B = Point has: (X2, Y2)) then:
120 
121               A + B = Point has: (X1 + X2, Y1 + Y2).
